# 📌 Project Overview

Completed the **Tata iQ GenAI Powered Data Analytics Job Simulation** on **Forage**, where I worked as an **AI Transformation Consultant** helping a financial services company (**Geldium Finance**) improve credit card delinquency prediction using **Generative AI**, **Predictive Analytics**, and **Responsible AI**.

The simulation focused on applying AI-assisted analytics to real-world business challenges by combining exploratory data analysis, predictive modeling, business recommendations, and AI strategy.

---

# 🎯 Business Problem

Geldium Finance experienced increasing **credit card delinquency rates**, leading to higher financial risk and inefficient manual collection processes.

The objective was to build an AI-powered framework capable of:

- Identifying customers likely to miss payments
- Supporting proactive intervention strategies
- Improving collections efficiency
- Ensuring fairness, transparency, and regulatory compliance

# 📂 Project Tasks

## Task 1 — Exploratory Data Analysis (EDA)

### Objective

Analyze customer credit data to identify patterns affecting delinquency prediction.

### Activities

- Reviewed dataset structure
- Identified missing values
- Detected outliers and anomalies
- Assessed data quality
- Evaluated customer risk indicators
- Suggested missing value treatment strategies
- Explored synthetic data generation for incomplete records
- Used GenAI tools for dataset summarization and insight generation

### Key Risk Indicators

- Previous missed payments
- High credit utilization
- High Debt-to-Income Ratio
- Income instability
- Recent credit activity

---

## Task 2 — Predictive Modeling Framework

### Objective

Design a no-code AI-powered predictive model for customer delinquency.

### Model Options Evaluated

- Logistic Regression
- Decision Trees
- Neural Networks

### Selected Approach

**Logistic Regression**

Reason:

- High interpretability
- Probability-based predictions
- Easy deployment
- Regulatory friendly
- Suitable for binary classification

### Key Input Features

- Payment History
- Credit Utilization
- Debt-to-Income Ratio
- Income
- Recent Credit Activity

### Model Evaluation Metrics

- Accuracy
- Precision
- Recall
- F1 Score
- ROC-AUC
- Confusion Matrix

### Responsible AI Checks

- Bias Detection
- Fairness Evaluation
- Explainability
- Transparency

---

## Task 3 — Business Recommendation Report

Prepared a stakeholder-ready report summarizing:

- Customer delinquency trends
- High-risk customer segments
- Business recommendations
- Ethical AI considerations

### SMART Recommendation

Implement proactive outreach for customers with:

- High credit utilization
- Multiple missed payments
- High DTI ratio

using automated reminders and personalized repayment support to reduce 30+ day delinquency.

---

## Task 4 — AI-Powered Collections Strategy

Designed a high-level autonomous collections framework.

### Proposed Workflow

```

        │

        │
        ▼
Continuous Learning & Monitoring
```
